# LLM Parallel Inference Engine
## UCS645 - Aakash Chandra | Roll No: 102317242

A GPT-style LLM inference engine built in C++ from scratch,
parallelized using OpenMP, MPI, and CUDA.

### Phases
- Phase 1: Serial baseline (C++)
- Phase 2: Shared memory (OpenMP)
- Phase 3: Distributed memory (MPI)
- Phase 4: Heterogeneous computing (CUDA)
